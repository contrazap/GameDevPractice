# Skill Matrix

The 17 reference games mapped to the skills we take from them and where those are covered. Games aren't cloned; their load-bearing mechanics become features.

| Game | What we take from it | Covered in |
|---|---|---|
| GTFO | Coop horror FPS loop: stealth vs hordes, terminals, bioscans, darkness/flashlight, resource scarcity, team revive | **P1 (entire phase)** — this is the September game |
| Project IGI | FPS gunplay, AI perception/stealth | P1.3, P1.6 |
| PUBG | Weapons feel, session/matchmaking flow, inventory | P1.3, P1.11, P3.3 |
| Dark Souls 2 | Melee: lock-on, stamina, dodge i-frames, combos, checkpoint design | P3.1 |
| Prince of Persia WW | Traversal/platforming, combat combos, touch-friendly controls | P3.2, P0.3 (touch) |
| Project Zomboid (3D, better coop) | Survival systems, needs, crafting, persistence, coop world state | P3.3, P3.4 |
| Don't Starve Together (3D topdown) | Survival coop, day/night cycle, harvesting | P3.4 |
