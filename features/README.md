# features/

Detailed feature plans land here, generated on demand — see [PLAN-GENERATOR.md](../PLAN-GENERATOR.md) for the invocation.

Naming: `<ID>-<short-slug>.md`, e.g. `P0.2-eos-plugin-port.md`, `P1.6a-sleeper-ai.md`.

When a plan is generated, the feature's status in [ROADMAP.md](../ROADMAP.md) moves ⬜ → 📋. Variation-challenge specs live alongside as `<ID>-variation.md`.
